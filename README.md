# HoYoTimer
ほよゲーのスタミナ管理ができるWindowsアプリ
原神・スターレイル・ゼンゼロに対応しています

x64 (Intel, AMDなど) 版はReleasesにあるwin_x64.zipをダウンロードしてください。Arm (Qualcommなど) は未対応です。

ファイルを展開し、フォルダ内にある`HoYoTimer.exe`ファイルを実行してください。

アプリの使い方やリリースノートは https://musenmy.github.io/ja/app/hoyotimer.html を参照。

<br>

Stamina management app for HoYo games runs on Windows

Supports Genshin, Star Rail, Zenless.

THIS APP IS AVAILABLE ONLY IN JAPANESE.

download zip files from Releases, then extract them and run `HoYoTimer.exe`.
only x64 devices are supported.
